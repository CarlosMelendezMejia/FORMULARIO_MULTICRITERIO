Instrucciones del Sistema
Descripción General: Este sistema permite priorizar factores mediante la asignación de valores únicos. Cada usuario recibe un formulario (posiblemente protegido con contraseña) y asigna una jerarquía de importancia a los factores disponibles. El administrador posteriormente pondera y genera un ranking global.
Pasos para Usuarios
Acceso al Formulario

Ingresa tu ID de Usuario y haz clic en Ingresar al Formulario. Si el formulario está protegido aparecerá una pantalla para introducir la contraseña; ésta se pedirá cada vez que abras el formulario.

Verificación de Datos

Verifica tus datos personales mostrados (nombre, apellidos, cargo, dependencia). Si falta algo complétalo.

Asignación de Prioridades

Asigna a cada factor un número único (sin repetir) dentro del rango permitido. El valor representa su prioridad relativa.

Revisión de Simbología

Revisa la tabla de simbología para entender la dimensión de cada factor.

Guardar Respuestas

Haz clic en Guardar Respuestas. Confirma en el cuadro de diálogo. El sistema bloqueará tu respuesta (ya no podrás editarla a menos que un administrador la reabra).

Confirmación

Observa la pantalla de confirmación y, si lo deseas, regresa al inicio.

Pasos para Administradores
Inicio de Sesión

Inicia sesión con la contraseña de administrador.

Gestión de Respuestas

En el panel principal filtra, pagina y exporta respuestas a CSV.

Gestión de Formularios

Gestiona formularios: creación, activación/desactivación de contraseña y eliminación (con confirmación segura).

Edición de Factores

Edita factores (nombre, descripción, color y dimensión) y agrega nuevos.

Ponderaciones

Asigna ponderaciones a cada respuesta; el ranking se recalcula (usa caché temporal).

Reapertura

Reabre respuestas para permitir que un usuario vuelva a modificar su priorización.

Exportación

Exporta el ranking consolidado a CSV cuando las ponderaciones estén completas.

Notas Importantes
  • Las contraseñas de formularios se almacenan como variables de entorno; cambiar una se refleja inmediatamente.
  • Si se reabre una respuesta deja de estar bloqueada y aparece como pendiente hasta completarse de nuevo.
  • El ranking sólo incluye respuestas completamente ponderadas y bloqueadas.
  • Si la aplicación se sirve bajo un subpath, las URLs se ajustan automáticamente.
